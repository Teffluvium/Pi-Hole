# Pi-Hole
Support scripts and configurations for Pi-Hole


## Updating the OS

```
sudo apt-get update
sudo apt-get -y upgrade

```

## Update the Python Environment
```
sudo apt-get install --upgrade python3-pip python3-venv python3-setuptools
```
